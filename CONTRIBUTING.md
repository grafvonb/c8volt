Thank you for helping improve `c8volt`.

not require a separate Contributor License Agreement.

## Developer Certificate of Origin

Contributions must include a Developer Certificate of Origin sign-off in each
commit message:

```text
Signed-off-by: Your Name <your.email@example.com>
```

The sign-off means that you certify:

- you wrote the contribution, or you otherwise have the right to submit it;
- the contribution may be distributed under this project's open source license;
- the contribution and its commit metadata may be maintained publicly as part of
  the project history.

You can add the sign-off automatically with:

```bash
git commit -s
```

## Trademarks

The `c8volt` name and logo are covered by [TRADEMARKS.md](./TRADEMARKS.md).
Modified versions must not present themselves as the official `c8volt` project.

## Contribution Quality

Please keep changes focused, documented, and tested. When a change affects
commands, configuration, generated documentation, release artifacts, licensing,
or security behavior, update the related docs in the same pull request.
