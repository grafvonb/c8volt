## Codebase Patterns
- Public `c8volt/<module>` facade methods belong in `client.go`, convert domain models in `convert.go`, and translate internal service errors with `c8volt/ferrors.FromDomain`.
- New `get` subcommands live in `cmd/get_<resource>.go`, register under `getCmd`, and should render single-item output through `cmd/cmd_views_get.go` so one-line, JSON, and `--keys-only` modes stay aligned.
- Generated single-object API responses can decode malformed `200 OK` JSON into non-nil zero-value structs, so versioned services should validate the converted domain object after `RequirePayload` when the endpoint guarantees one item.
